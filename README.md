# Java-Chat-Box
Chat Application is made using graphical components in the Swing toolkit in Java with MySQL Database. The feature of this chat application includes sign in, sign up, chatting, creating and joining connections. Here you can create your own account and chat freely. This whole system is built using Java programming language with the help of Netbeans IDE. In order to create an account, the user has to enter a username, id, and password. After this, the application will take the user toward a chat section.  Likewise, for creating a connection, you have to enter chat name, select type, and password. And to join a connection you just have to select a connection type, chat name and password. The application connects to a MySQL database that holds users record.
